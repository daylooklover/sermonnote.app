📘 말씀노트 사용 가이드

🌟 말씀노트의 독창성과 장점

  • AI 설교 요약/보완 기능: 설교 초안을 자동 요약, 문법 교정, 주제에 맞는 성경 구절 자동 삽입
  • 음성 기반 설교 작성: 설교를 말하면 실시간 텍스트 변환, 아이디어를 손쉽게 기록
  • 퀵메모 연동 시스템: 즉흥적인 영감을 퀵메모로 저장 후 설교 작성 화면에서 불러오기
  • 강해설교 주석 기능: 성경 원어 분석, AI 주석 제안, 묵상 적용 주석 추가 가능
  • 맞춤식 설교 생성: 청중 연령, 상황, 주제, 톤에 따라 AI가 설교 전체를 구조화 생성
  • Firebase 클라우드 저장: 실시간 동기화 및 안전한 저장
  • PDF 출력 & 설교 공유: 버튼 클릭으로 PDF 저장 및 링크 공유

🛠 사용법

  1. 홈화면에서 Get Started 클릭
  2. 로그인 후, 설교 작성 메뉴로 이동
  3. 필요시 음성 입력 또는 AI 버튼 클릭
  4. 퀵메모는 우측 상단 버튼 클릭으로 접근 가능
  5. 작성 완료 후 PDF 다운로드 또는 링크로 공유

✝️ 말씀을 전하는 사역자들에게 최고의 도구가 되기를 기도합니다.
